About {{ g.domain_capitalized }}

{{ g.domain_capitalized }} is a free torrent search engine that offers you fast, simple and easy access to every torrent in one place. Torrents.com crawls millions of Internet addresses to automatically generate meaningful links to your queries.

We are not a torrent tracker. We are not a search aggregator.

{{ g.domain_capitalized }} does not host, does not reproduce nor publish the linked torrents that appear as search results. Nevertheless, if you believe any of the linked works are being disseminated without the appropriate authorisation from the rights holders, or that they infringe the rights of third parties, you may request that we remove them using the copyright claim form.

You should read and accept our terms of use and our privacy policy before using the services provided by {{ g.domain_capitalized }}.

{{ g.domain_capitalized }} uses these amazing technologies:

  • Foofind: Foofind is an open source universal search engine.
  • Flask: Flask is a microframework for Python based on Werkzeug, Jinja 2 and it's BSD licensed.
  • MongoDB: MongoDB (from "humongous") is a scalable, high-performance, open source NoSQL database.
  • Sentry: Sentry is a realtime event logging and aggregation platform. It specializes in monitoring errors and extracting all the information needed to do a proper post-mortem without any of the hassle of the standard user feedback loop.
  • Sphinx: Sphinx is an open source full text search server, designed from the ground up with performance, relevance (aka search quality), and integration simplicity in mind. It's written in C++ and works on Linux (RedHat, Ubuntu, etc), Windows, MacOS, Solaris, FreeBSD, and a few other systems.
  • jQuery: jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.
  • jQuery plugin: Treeview: Lightweight and flexible transformation of an unordered list into an expandable and collapsable tree, great for unobtrusive navigation enhancements.
